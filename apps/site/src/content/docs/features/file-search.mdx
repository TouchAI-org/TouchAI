这一页讲的是 TouchAI 如何通过 `file_search` 和 `read` 两个工具，继续处理本地资料。

## 文件搜索

`file_search` 用来快速定位本地文件。

- 找资料
- 找最近保存的文档
- 找截图、PDF、表格、代码文件
- 在大量文件里先缩小目标范围

## 读取文件

`read` 用来继续打开文件内容，让 TouchAI 在找到文件后继续整理、总结和提问。

常见可处理内容包括：

- 文本
- 代码文件
- 图片
- PDF
- 目录结构

## 常见用法

- 找文件 → 读内容 → 总结重点
- 找资料 → 提炼信息 → 继续追问
- 找目录 → 确认范围 → 再读取具体内容
