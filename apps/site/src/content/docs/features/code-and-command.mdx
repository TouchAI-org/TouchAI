这一页讲的是 TouchAI 如何通过 `apply_patch` 和 `bash` 两个工具，继续处理代码和本地任务。

## 修改文件

`apply_patch` 直接修改本地工作区文件。

- 改文案
- 调整代码
- 修改配置
- 在现有文件上继续推进任务

## 执行命令

`bash` 执行本地 shell 命令，并返回输出结果。

- 运行构建、测试和脚本
- 查看 Git 状态和日志
- 安装依赖
- 排查错误

:::warning[注意]
执行命令会经过审批机制。建议先确认命令用途，再继续执行。
:::

## 常见用法

- 看代码 → 修改文件 → 生成补丁
- 执行命令 → 查看结果 → 继续处理错误
- 改配置 → 运行验证 → 再继续调整
