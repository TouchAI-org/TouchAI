## 下载

最简单的开始方式，是从 [GitHub](https://github.com/TouchAI-org/TouchAI) 下载最新版本。

1. 访问 [Releases 页面](https://github.com/TouchAI-org/TouchAI/releases)
2. 选择最新版本，在 **Assets** 区域查看安装包
3. 根据你的系统下载对应文件：
   - **Windows**：选择 `.exe` 或 `.msi`
   - **macOS**：选择 `.dmg`
   - **Linux**：选择 `.deb`、`.rpm` 或 `.AppImage`
4. 下载完成后，双击运行安装程序，并按照提示完成安装
