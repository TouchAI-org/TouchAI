TouchAI 的核心体验，不是“打开一个 AI 应用”，而是**在当前工作流里直接开始使用 AI**。

按照下面几步，你可以在几分钟内完成第一次成功体验。

## 1. 配置模型

第一次使用前，你需要先添加一个可用的 AI 提供商和模型。

你可以这样开始：

1. 在电脑右下角隐藏托盘里找到TouchAI图标
2. 右键打开菜单栏选择设置
3. 进入服务商与模型
4. 添加服务商并填写必要信息
5. 刷新模型列表并选择默认模型

如果你需要详细配置说明，请查看[模型配置](/settings/model-configuration/)。

![服务商与模型设置页面](/getting-started/quickstart/quickstart-provider-settings.png)

## 2. 发起第一次对话

配置完成后，就可以开始第一次实际使用：

1. 按 `Alt+Space` 唤起 TouchAI
2. 直接输入你的问题、任务或需求
3. 按 Enter 发送
4. 等待 AI 返回结果

![第一次对话示意](/getting-started/quickstart/quickstart-first-chat.png)

## 4. 更顺手的使用方式

TouchAI 最适合接住你手头正在做的事情，而不是把它当成一个独立的问答框。

- 想到问题就直接唤起，不用先切换到别的应用
- 一个任务尽量放在同一个会话里继续推进
- 先让它帮你处理“中间步骤”，例如提炼重点、整理结构、改写表达
- 从真实任务开始体验，而不是刻意测试 AI
