如果你在使用 TouchAI 的过程中遇到问题，可以先从这里开始排查。

## 安装后无法启动

你可以先检查：

1. 是否安装了最新版本
2. 应用是否已经在系统托盘中运行
3. 是否有其他应用占用了默认快捷键 `Alt+Space`
4. 是否需要重新启动应用或系统
5. 是否能在日志中看到明显报错

如果你还没有完成安装流程，可以先查看[安装指南](/getting-started/installation/)。

## 无法开始对话

最常见的原因是：还没有配置可用的模型。

你可以先确认：

1. 是否已经添加了服务商
2. 是否已经填写正确的 API Key 或接口地址
3. 是否已经成功刷新模型列表
4. 是否已经选择默认模型

详细说明请查看[模型配置](/settings/model-configuration/)。

## API 调用失败

你可以先检查：

1. 网络连接是否正常
2. API Key 是否填写正确
3. 账户额度或余额是否足够
4. 提供商接口是否可用
5. 日志中是否有具体错误信息

## 工具执行超时

如果某些任务执行时间较长，可以先尝试：

1. 调整对应工具的超时时间设置
2. 缩小任务范围，减少一次性处理的内容
3. 优化输入内容，让任务更明确
4. 检查本地环境或网络状态

## 快捷键没有反应

如果按下快捷键没有唤起 TouchAI，可以先检查：

1. TouchAI 是否已经启动
2. 当前快捷键是否与其他应用冲突
3. 是否在设置中修改过快捷键
4. 重新设置一个更少冲突的组合键后再试一次

你也可以查看[快捷键](/getting-started/shortcuts/)了解更多操作方式。

## 获取帮助

如果上面的排查还不能解决问题，可以：

1. 在 [GitHub Issues](https://github.com/TouchAI-org/TouchAI/issues) 中搜索类似问题
2. 提交新的 Issue，并附上：
   - 操作系统版本
   - TouchAI 版本
   - 错误日志
   - 复现步骤
3. 在 [GitHub Discussions](https://github.com/TouchAI-org/TouchAI/discussions) 参与讨论
