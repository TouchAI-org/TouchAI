MCP 扩展可以让 TouchAI 接入更多外部服务，使得他的功能不再局限于官方预设的服务。通过自行导入各类 MCP 服务，你可以随心所欲地定制 TouchAI 的能力版图。

## MCP 能为你做什么？

面对用户个性化的需求，固定的功能往往难以招架。MCP 扩展可以让你你根据实际场景，灵活接入所需的外部资源：

- **自由导入，按需连接**：支持用户自行导入各类 MCP 服务，无论是常用网站还是小众工具，都能轻松接入。
- **打破边界，无限可能**：不再受限于内置功能，你可以随时为 TouchAI 拓展访问外部数据和调用新能力的权限。
- **量身定制，直达任务**：根据你的工作流灵活组装服务，将简单的对话直接转化为专属的、可落地的真实任务。
