---
title: 添加服务
description: 在 TouchAI 中添加并管理一个 MCP 服务
---

如果你想开始使用 MCP 扩展，可以直接在 **MCP 工具** 页面添加服务。

## 1. 打开 MCP 工具页

进入应用 **设置**，在左侧导航中打开 **MCP 工具**。

## 2. 新建服务器

点击左下角的 **+ 添加 MCP 服务器**。

创建后，左侧会出现新的服务项；右侧可以在 **配置 / 工具 / 日志** 三个页签中查看和管理它。

## 3. 在页面里完成配置

通常你只需要：

- 填写服务名称
- 选择传输类型
- 填写服务需要的必要信息
- 保存并检查连接状态

## 4. 查看工具和日志

配置完成后：

- 在 **工具** 页签中查看这个服务提供了哪些能力
- 在 **日志** 页签中查看连接、调用和报错记录

当状态显示为 **已连接** 时，对应能力就可以在对话中使用。
