title: 热门服务
description: 常见 MCP 服务及其典型用途
---

我们为你甄选了覆盖高频场景的实用 MCP 服务。你可以自由导入这些服务，让 TouchAI 无缝接入你的真实工作流。

## 滴答清单
将对话直接转化为行动，高效管理你的时间与待办。

- 通过对话直接创建今日待办，无需手动输入
- 自动整理任务优先级，帮你聚焦最重要的事项
- 将复杂的长周期任务拆分为可执行的子步骤
- 一键回顾近期安排，清晰掌握工作与生活节奏


## 文件系统
打通 TouchAI 与你本地设备的壁垒，让 AI 读懂你的资料。

- 直接读取电脑本地的文档、笔记与参考资料
- 快速归纳长篇文档的核心要点，辅助写作与总结
- 在复杂的目录结构中快速定位你需要的文件
- 基于文件夹内的实际内容，回答针对性的问题

## 浏览器自动化

- 指挥 AI 打开指定网页，模拟人工访问路径
- 精准提取网页中的表格、文章或特定数据
- 自动执行登录、填写表单、点击等重复操作
- 将分散在多个网页的信息汇总成结构化的结果

## Git

- 快速查看代码差异（Diff），理解具体的修改内容
- 自动梳理提交记录（Commit Log），还原开发脉络
- 读取项目结构与配置文件，快速上手新仓库
- 辅助分析版本迭代变化，生成清晰的更新说明

## 团队沟通工具 (如钉钉/飞书等)

- 获取群组或私聊的最新消息，不错过重要通知
- 让 AI 生成的报告或提醒，自动发送到指定频道
- 实时更新任务状态或工作进度，保持团队信息一致
- 自动提炼群聊讨论的重点，生成会议纪要或待办

## 数据库 / 业务系统

- 无需编写 SQL，直接用对话查询数据库中的数据
- 自动拉取多张数据表，生成综合性的业务报表
- 基于历史业务数据，辅助进行简单的趋势预判与分析
- 支持查询订单、库存等日常业务系统的实时状态

## 淘宝

- 用自然语言描述需求，快速定位目标商品
- 自动整理商品规格、参数等关键信息
- 将多个候选商品放在一起，直观比较优劣
- 根据你的预算和偏好，提供理性的比价与筛选建议
