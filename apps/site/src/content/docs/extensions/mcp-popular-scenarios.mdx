title: 热门服务
description: 常见 MCP 服务及其典型用途
---

下面这些是比较常见、也更容易理解用途的 MCP 服务。

## 滴答清单

- 查看今天的待办
- 整理任务优先级
- 拆分复杂任务
- 汇总近期安排

## 淘宝

- 搜索商品
- 整理商品信息
- 比较多个候选项
- 辅助比价和筛选

## 文件系统

- 读取本地资料
- 整理文档
- 查找文件
- 从目录内容继续提问

## 浏览器自动化

- 打开网页
- 提取页面内容
- 执行重复网页操作
- 把网页信息整理成结果

## Git

- 查看改动
- 理解提交记录
- 整理项目信息
- 辅助梳理版本变化

## 团队沟通工具

- 读取消息
- 发送通知
- 同步状态
- 汇总沟通内容

## 数据库 / 业务系统

- 查询数据
- 汇总结果
- 辅助分析
- 支持日常业务处理
