---
title: 获取 API Key
description: 常用模型提供商的 API Key 获取方式
---

如果你准备配置模型，可以先到对应服务商平台创建 API Key。

## OpenAI

1. 打开 [OpenAI Platform](https://platform.openai.com)
2. 登录或注册账号

![OpenAI Platform 首页](/settings/api-keys/OpenAI/openai-platform-home.png)

3. 进入 **API Keys** 页面

![OpenAI API Keys 页面](/settings/api-keys/OpenAI/openai-api-keys-page.png)

4. 点击创建新的 API Key

![OpenAI 创建 API Key 对话框](/settings/api-keys/OpenAI/openai-create-key-dialog.png)

5. 复制保存，并填写到 TouchAI 中


## Anthropic（Claude）

1. 打开 [Anthropic Console](https://console.anthropic.com)
2. 登录或注册账号
3. 进入 **Settings / API Keys**
4. 创建新的 API Key
5. 复制保存，并填写到 TouchAI 中

- [https://console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys)

## Google Gemini

1. 打开 [Google AI Studio](https://aistudio.google.com)
2. 登录 Google 账号
3. 进入 **Get API Key** 页面
4. 创建新的 API Key
5. 复制保存，并填写到 TouchAI 中

- [https://aistudio.google.com/apikey](https://aistudio.google.com/apikey)

5. 复制保存，并填写到 TouchAI 中

- [https://platform.deepseek.com/api_keys](https://platform.deepseek.com/api_keys)
