在 **服务商与模型** 页面中，你可以添加模型提供商、填写 API Key、刷新模型列表，并选择默认模型。

## 支持的提供商

TouchAI 支持主流模型提供商，也支持兼容 OpenAI 接口的服务。

常见选择包括：

- OpenAI
- Anthropic（Claude）
- Google（Gemini）
- DeepSeek
- 智谱 AI
- Moonshot（Kimi）
- 阿里云通义千问
- xAI（Grok）
- 自定义 OpenAI 兼容接口

## 添加一个提供商

你可以按下面步骤完成配置：

1. 打开 **设置**
2. 进入 **服务商与模型**
3. 添加一个新的服务商
4. 填写服务商名称、接口地址和 API Key
5. 保存后刷新模型列表
6. 选择一个默认模型

![服务商与模型页面](/settings/model-configuration/model-configuration-overview.png)

配置完成后，就可以在 TouchAI 中直接使用该模型。

## 获取 API Key

常见服务商的获取步骤已经单独整理到：

- [获取 API Key](/settings/api-keys/)

## 默认模型选择

在模型列表里刷新出可用模型后，选择一个作为默认模型即可。

如果你只是先开始使用，可以先选一个你最常用、最稳定的模型，后续再根据任务切换。

## 模型切换

在日常使用中，你可以直接在对话界面切换模型：

1. 点击输入框上方的模型图标
2. 或使用快捷键 `Ctrl+M`
3. 从列表中选择当前要使用的模型

## 安全说明

API Key 只保存在本地，不会上传到 TouchAI 服务器。

填写密钥时，建议：

- 不要把密钥分享给他人
- 优先保存在自己的账号体系中
- 如果怀疑泄露，及时到对应平台更换

## 常见问题

### API Key 填错时

进入 **服务商与模型** 页面，编辑对应服务商，把错误信息改正后重新保存即可。

### 多个提供商

可以。你可以同时添加多个提供商和多个模型，并在使用时自由切换。

### 模型列表刷新不出来

通常可以优先检查：

1. 接口地址是否正确
2. API Key 是否有效
3. 网络连接是否正常
4. 提供商接口是否可用
