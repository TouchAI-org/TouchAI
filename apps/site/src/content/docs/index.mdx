## TouchAI 是什么

TouchAI 是一个桌面 AI 助手，强调 **“一触即达、不中断当前工作流”**。

你可以通过快捷键快速唤起它，在当前任务里直接提问、整理资料、处理文件、继续思考，而不用频繁切换窗口或把上下文重新讲一遍。

## 技术栈

- **前端框架**: Vue 3.5 + Vite 8 + TypeScript
- **UI 库**: Tailwind CSS 4 + Reka UI + TipTap
- **状态管理**: Pinia
- **后端框架**: Tauri 2 (Rust)
- **数据库**: SQLite + Drizzle ORM
- **AI SDK**: @ai-sdk (支持 OpenAI, Anthropic, Google 等)
- **包管理器**: pnpm
